Laboratório Conectado

Este projeto teve como objetivo estruturar um laboratório de informática com computadores conectados à internet via cabo, garantindo uma conexão estável e eficiente para o uso acadêmico. Participei de todas as etapas, desde a montagem física das máquinas, passando pela instalação de sistemas operacionais e softwares, até o gerenciamento e configuração da rede local.

Esse trabalho me proporcionou uma experiência profunda no gerenciamento de infraestrutura de TI, otimizando o ambiente para facilitar o aprendizado dos alunos e o trabalho dos professores.
